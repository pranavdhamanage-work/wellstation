Project Setup Instructions

Virtual Environment Activation

To activate the virtual environment, follow these steps:

1. Open your terminal/command prompt
2. Navigate to the project directory
3. Run the following command:

For Windows users (git bash):
. .\python-venv\Scripts\activate

pip install -r requirements.txt

Additional Notes
- Ensure Python is installed on your system
- Verify the virtual environment activation by checking if the terminal prompt shows the environment name
- To deactivate the environment, simply type `deactivate`
